StrapCharge é uma pulseira de silicone com células solares flexíveis embutidas que captam luz do sol e de ambientes internos, acumulando energia em uma bateria integrada. Conecta ao celular via cabo retrátil escondido na própria fivela, sem precisar carregar nada extra.

Problema que resolve

Celular morrendo na rua, sem tomada por perto — algo que todo mundo já passou. A StrapCharge resolve isso de forma passiva, sem que o usuário precise fazer nada além de usar a pulseira normalmente.

Diferenciais

Carrega passivamente enquanto você anda. Cabo retrátil embutido na fivela com conector USB-C e Lightning. Sem bateria externa para carregar e sem cabos para guardar. Leve e discreta, com aparência de smartband comum.

Público-alvo

Pessoas ativas, viajantes e estudantes — quem fica o dia todo fora de casa e depende do celular.

Versões

StrapCharge padrão — cabo retrátil, bateria integrada, células solares flexíveis. Preço entre R$ 180 e R$ 250.

StrapCharge Pro — bateria com maior capacidade, NFC para pagamentos por aproximação e mais autonomia energética.

Site

O site do produto foi desenvolvido em HTML, CSS e Bootstrap 5, com navbar responsiva, carrossel de imagens e footer. Para rodar localmente, basta abrir o arquivo index.html no navegador — não precisa instalar nada.

Estrutura de pastas

index.html é a página principal. A pasta css contém o arquivo style.css com os estilos personalizados. A pasta img contém as imagens do produto e o logo.
